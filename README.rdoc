== Welcome to Treebook

Treebook is a ruby on rails web application that emulates major features from major social networks. Most of the code is from the teamtreehouse.com class on ruby with minor orginial tweaks and enhancements.


== Getting Started

1. Visit http://pure-citadel-7483.herokuapp.com/ to access the latest deployment of the app.

2. Register for an account.

3. Post a status.

4. Enjoy!
